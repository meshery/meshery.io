extensionInfo: |
  Shape Builder is a Meshery extension that provides an interactive, visual canvas for designing custom polygon shapes used to represent Meshery Components in Kanvas. Users can draw polygons on a snap-to-grid canvas and export normalized SVG coordinate notation ready for use in Meshery Component definitions.
extensionCaveats: |
  - Interactively draw custom polygon shapes on a snap-to-grid canvas with point-by-point control
  - Export normalized SVG matrix coordinates ready for use in Meshery Component polygon shape definitions
  - Scale shapes using preset multipliers or a continuous slider, with real-time coordinate updates
  - Keyboard shortcuts for closing shapes (Enter/Esc), undoing points (Ctrl+Z), and fine-grained snapping (Ctrl)
