Empowering Open Source Contributors

Meshery participates in the Linux Foundation Mentorship (LFX) program, offering internship opportunities across multiple terms each year. The LFX Mentorship program connects aspiring developers with experienced open source communities to build real-world skills and make meaningful contributions to cloud native infrastructure.

Browse our programs by year to see available projects, learn about program timelines, and discover how you can participate in shaping the future of cloud native technology.
