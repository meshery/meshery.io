# 🧱 Meshery Projects

Meshery is a powerful cloud native management plane that includes multiple components. Each component is an individual project or repository in the Meshery ecosystem.

---

## 🔧 Meshery CLI

A command-line tool for interacting with Meshery.

➡️ [View on GitHub](https://github.com/meshery/meshery)

---

## 🧩 Meshery Adapters

Adapters help Meshery integrate with service meshes like:

- Istio
- Linkerd
- Consul
- Kuma
- Open Service Mesh (OSM)

➡️ [View Adapters](https://github.com/meshery)

---

## 💻 Meshery UI

The React-based frontend of Meshery.

➡️ [UI Repository](https://github.com/meshery/meshery)

---

## ⚙️ MeshKit

A utility library used to standardize and simplify service mesh tools.

➡️ [MeshKit on GitHub](https://github.com/meshery/meshkit)

---

## 📦 Meshery Operator

A Kubernetes controller to install and manage Meshery.

➡️ [Operator on GitHub](https://github.com/meshery/meshery-operator)

---

## 📊 Service Mesh Performance (SMP)

A specification and toolset for benchmarking and reporting service mesh performance.

➡️ [SMP GitHub](https://github.com/service-mesh-performance)

---

## 🔬 Nighthawk

Performance testing tool, used alongside SMP.

➡️ [Nighthawk GitHub](https://github.com/envoyproxy/nighthawk)

---

*Explore and contribute to any of these Meshery components!*
