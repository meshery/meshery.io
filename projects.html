Meshery Community Projects

Welcome to the Meshery Community Projects Handbook! Here you’ll find information about the various projects under the Meshery umbrella, as well as resources and guides for contributors.

Active Projects

Meshery

The open source, cloud native management plane for service meshes and Kubernetes.

Meshery Operator

Kubernetes operator for managing Meshery deployments.

Meshery UI

The user interface for Meshery, built for extensibility and ease of use.

Meshery Adapters

Integrations for different service meshes (Istio, Linkerd, Consul, etc.).

MeshMap

Visual designer for cloud native infrastructure and service mesh topologies.
