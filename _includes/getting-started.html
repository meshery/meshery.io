Brew User:

Install on Mac or Linux using Homebrew:

$ brew install mesheryctl $ mesheryctl system start {% include copy-to-clipboard.html clipboard_target="#start-brew" clipboard_text="brew install mesheryctl mesheryctl system start" %}

Docker User:

Get mesheryctl

Install mesheryctl with Bash, Brew, Scoop, or download directly.

Use mesheryctl

mesheryctl system context create docker --platform docker --set mesheryctl system start {% include copy-to-clipboard.html clipboard_target="#start-helm-v3" clipboard_text="mesheryctl system context create docker --platform docker --set mesheryctl system start" %}

Use Docker Extension for Meshery

Windows User:

Using Scoop

$ scoop bucket add mesheryctl https://github.com/meshery/scoop-bucket.git $ scoop install mesheryctl $ mesheryctl system start {% include copy-to-clipboard.html clipboard_target="#start-windows" clipboard_text="scoop bucket add mesheryctl https://github.com/meshery/scoop-bucket.git scoop install mesheryctl mesheryctl system start " %}

Using Binary

Not a Scoop user? Download and unzip the mesheryctl binary directly from the Meshery releases page. Add mesheryctl to your path for ease of use. Then, execute:

$ mesheryctl system start {% include copy-to-clipboard.html clipboard_target="#start-windows" clipboard_text="mesheryctl system start" %}
